INSTRUCCIONES DE ENTREGA
------------------------
Este .zip contiene una plantilla mínima de currículum web para usar como entrega de la práctica.
Archivos incluidos (raíz del zip):
 - index.html
 - styles.css
 - script.js
 - README.txt

Cómo usar:
 1. Descomprime el .zip.
 2. Abre index.html en tu navegador (doble clic).
 3. Para servirlo con un servidor local (recomendado), puedes usar:
    python3 -m http.server 8000
    y abrir http://localhost:8000 en tu navegador.

Consejos para completar:
 - Personaliza el contenido (nombre, email, enlaces a repos y descripciones).
 - Añade imágenes en una carpeta 'assets' e inclúyelas desde index.html.
 - Si usas recursos externos (fuentes, iconos), súbelos junto al sitio o usa enlaces absolutos.
 - Asegúrate de que el zip final entregue sólo los ficheros que pide la práctica (HTML/CSS/JS).

Referencia:
 - Sigue las guías de HTML/CSS/JS que te pide la práctica para entender cada parte (ej.: W3Schools).
